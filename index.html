DAY-2 Interview Questions

1. Explain br tags

The br tag in HTML is used to create a line break in a text. It is an empty tag which means it does not need an end tag.

2. Explain how many different types of lists we have in HTML

We have 3 types of list present in our HTML
1. UnOrdered List- starts with thye ul tag and each list item starts with the li tag. The list items will be marked with bullets by default.
2. Ordered List- starts with thye ol tag and each list item starts with the li tag. The list items will be marked with numeric by default
3. Description list- dl tag defines the description list , the dt tag defines the term(name) and dd tag describes each term

3. What are semantic tags

Semantic HTML tags are tags that define the meaning of the content they contain. For example, tags like "header", "article", and "footer" are semantic HTML tags. They clearly indicate the role of the content they contain.
